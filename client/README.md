# Remindly Client

React frontend for Remindly.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Scripts

| Command           | Description              |
| ----------------- | ------------------------ |
| `npm run dev`     | Development server       |
| `npm run build`   | Production build         |
| `npm run preview` | Preview production build |
| `npm run lint`    | ESLint                   |
| `npm run format`  | Prettier                 |
